# 样品业务板块能力和 PDF 差距

> 盘点日期：2026-06-17  
> 对照来源：`docs/reference/远景外贸业务管理系统产品介绍-含仓库管理.pdf` 第 17-20 页 `1.5.3 样品管理`，第 49 页确认样/大货样实际日期读取说明，以及当前代码、测试和检查记录。

## 结论

样品业务板块的主干功能已经完成，覆盖 PDF 中的打样管理、样品登记、寄样管理、寄样统计和寄样费用统计。业务员可以从客户打样要求开始，维护打样进度和打样费用，把已完成打样转成样品台账，再登记样品图片、货号、来源和数量，最后创建寄样单、提交审核、审核扣减留样数量，并查询寄样历史、费用统计和寄样统计。

当前仍不能称为“PDF 级完全完成”的部分主要是体验和外部集成增强：标准 Excel/PDF 模板、真实快递轨迹接口、通用审批流替换、图片上传预览的一体化体验、多套自定义视图和更完整的批量导入导出模板。

## 当前能做什么

### 打样管理

- 新增打样单，记录客户、产品、供应商、业务员、打样日期、要求完成日期、打样去向和客户打样要求。
- 把客户打样要求拆成内部打样单明细，支持产品编号、产品名称、规格、数量、单位和明细要求。
- 查询打样单，支持关键字、状态、客户和打样日期范围过滤。
- 维护打样进度，支持内部打样、外发工厂、客户确认、完成等阶段，并同步打样单状态。
- 登记打样费用，记录费用类型、金额、币种、收款方和备注。
- 发起打样费支付流程，系统会生成合作伙伴费用发票和费用付款申请，并把财务发票号、付款申请号回写到打样费用。
- 打印内部打样单，后端生成 HTML 打印视图，前端可直接打开浏览器打印。
- 已完成打样可以转成样品登记，自动带出来源、客户、产品、供应商、数量和说明，并生成收样数量事件。

### 样品登记

- 维护样品台账，覆盖来样、确认样、大货样、留样等分类。
- 记录产品、客户、供应商、客户货号、供应商货号、采购合同、来源单据、收样日期、提交日期、数量、单位和说明。
- 登记样品图片元数据，新增样品时可带首图，样品明细中可追加图片。
- 自动维护样品数量台账：收样数、寄样数、公司留样数量。
- 支持收样/寄样数量事件，寄样事件可记录寄样单号、接收方和备注。
- 确认样、大货样登记提交日期后，会生成跟单节点事件；采购跟单接口可消费这些事件并回写实际日期。
- 支持按关键字、样品分类、客户、采购合同查询样品。
- 支持按当前表单导入样品，以及按筛选条件导出样品台账 CSV。

### 寄样管理

- 新增寄样单，记录客户、供应商、工厂、收件人、收件公司、收件地址、快递公司、快递单号、报价单和备注。
- 寄样单包含样品明细和费用明细，可关联样品台账、样品分类、产品、数量和单位。
- 草稿寄样单可编辑，已提交或已审核后禁止直接改草稿内容。
- 支持提交审核、审核通过；审核通过时会校验样品留样数量，库存不足会阻止审核。
- 审核通过后自动写入样品寄样数量事件，扣减公司留样数量。
- 支持物流跟踪字段维护：快递公司、快递单号、物流状态。
- 查询寄样单，支持关键字、状态、客户、快递公司和寄样日期范围过滤。
- 支持查看某个样品的寄样历史，也支持按客户和产品查询报价相关寄样历史。
- 支持寄样费用统计，按客户、日期和快递公司统计费用。
- 支持寄样统计，按状态、客户、快递公司统计寄样单数和寄样数量。
- 支持按筛选条件导出寄样 CSV。

## 和 PDF 还差哪些内容

| PDF 诉求 | 当前状态 | 仍需补齐 |
| --- | --- | --- |
| 打样要求转内部打样单并发送打样间/外发工厂 | 主干已完成 | 当前是系统内记录和打印；还未接邮件/消息发送、外部工厂协作门户 |
| 按打样日期、客户查询打样单和进度 | 已完成 | 可继续增加供应商、业务员、产品分类等高级筛选 |
| 打样费用查询和打样费支付流程 | 已完成主干 | 已生成正式费用发票/付款申请；后续可接完整审批流、付款状态自动反写 |
| 内部打样单打印/导出 | 部分完成 | 已有 HTML 打印；还缺标准 PDF/Excel 模板、模板配置和批量导出 |
| 样品资料图文并茂 | 部分完成 | 有图片元数据和图片上传 API；样品页面还缺文件选择器、缩略图预览、多图排序和附件化管理 |
| 样品收样数、寄样数、留样数自动计算 | 已完成 | 后续可增加盘点、报废、借样、归还等更多库存事件 |
| 查询样品来源、货号和相关单据 | 已完成基础 | 已记录来源和采购合同；还可增加来源单据弹窗跳转、报价/采购/寄样全链路反查 |
| 样品与寄样管理关联 | 已完成 | 寄样审核会扣减样品留样，样品明细可看寄样数量事件 |
| 寄样历史参照和寄样审核机制 | 已完成主干 | 当前是样品模块内提交/审核；后续可替换为通用审批流、加入撤回/驳回/作废 |
| 寄样跟踪管理 | 部分完成 | 已维护快递公司、单号和状态；还未对接 DHL/FedEx/UPS 等真实物流轨迹接口 |
| 寄样明细和寄样费用多条件统计 | 已完成基础 | 已按客户、日期、快递公司、状态统计并可导出 CSV；后续可进入报表中心、支持图表和下钻 |
| 导入导出 | 部分完成 | 样品台账和寄样支持 CSV 内容导出；还缺标准 Excel 模板、导入预校验报告和大批量异步导出 |
| 多视图/自定义视图 | 未专项完成 | 样品业务当前是固定列表/详情工作台；还缺保存多套列视图、筛选方案和用户级视图配置 |
