# 020 货物入库和库存台账检查记录

## PDF 对照

- `docs/05-实现步骤.md 6.3 货物入库`：入库单列表、从入库计划生成入库单、入库明细登记、待检入库、正式入库、入库审批、供应商入库记录查询。
- `docs/02-模块设计.md 仓库管理`：正式入库需校验商品、数量与计划一致，并且 QC 通过；所有库存变化必须写库存流水；为采购跟单提供入库完成日期。
- `docs/04-数据模型草案.md inbound_order / inventory_balance / inventory_ledger`：入库单、库存余额和库存流水。

## 本次实现范围

后端：

- `GET /api/v1/warehouse/inbound-orders`：入库单列表，支持文本、状态、入库模式、供应商、采购合同过滤。
- `POST /api/v1/warehouse/inbound-orders/from-plan`：从入库计划生成入库单，空明细时默认按待入库清单生成。
- `POST /api/v1/warehouse/inbound-orders/{order_id}/submit`：提交入库审批。
- `POST /api/v1/warehouse/inbound-orders/{order_id}/approve`：审批入库。
- `GET /api/v1/warehouse/inbound-orders/inventory-balances`：库存余额查询。
- `GET /api/v1/warehouse/inbound-orders/inventory-ledgers`：库存流水查询。
- 待检入库增加待检库存，不增加可用库存。
- 正式入库要求最近一次 QC 通过，并要求商品名称、数量与待入库清单一致。
- 正式入库后回写采购合同已收数量、入库计划已入库数量/关闭状态、采购跟单入库节点。
- SQLAlchemy async model：`InboundOrder`、`InboundOrderLine`、`InventoryBalance`、`InventoryLedger`。
- Repository 返回 typed row，封装单据、明细、余额、流水查询与写入。
- Service 负责权限、QC 拦截、计划一致性校验、库存记账和跨模块回写。
- FastAPI 路由显式声明 `response_model=ApiResponse[...]`。
- 权限：`warehouse:inbound_order:view/edit/approve/view_all`。

前端：

- 左侧菜单增加“货物入库”。
- 页面支持入库单列表、状态筛选、入库模式筛选、供应商筛选、采购合同筛选。
- 支持从入库计划生成正式入库或待检入库单。
- 支持提交审批和审批入库。
- 明细区展示采购合同、供应商、入库模式、状态、入库日期、库位和入库商品。
- 同页展示库存余额和库存流水，可按商品/仓库搜索。
- 选择入库计划时自动带出仓库、库位和经办人。

React 迁移补充：

- `frontend/src/App.tsx` 新增 `/warehouse/inbound-orders` React 路由和 `InboundOrdersPage`，替代占位模块。
- 复用 Ant Design 表格、表单、按钮、提示和现有 ERP 工作台布局，保持入库业务的高密度、可扫描工作区。
- 新增入库单状态、入库模式、库存状态和库存流水方向中文标签映射。
- 新增从入库计划生成、提交审批、审批入库、库存余额查询和库存流水查询的 payload 转换与 API 调用。
- 页面保留 E2E 依赖的字段 ID：`inbound-order-plan-id`、`inbound-order-code`、`inbound-order-mode`、`inbound-order-date`、`inbound-order-warehouse-id`、`inbound-order-location-name`、`inbound-order-reviewer`、`inbound-order-approved-at` 等。
- `frontend/src/styles.css` 新增入库单概览条、双栏工作区、详情区表格横向保护和移动端单列适配。
- 入库单生成、提交和审批成功提示均在数据刷新后展示，避免 E2E 或用户看到旧详情的刷新竞态。

## 可用性检查

- 列表、生成、提交、审批、明细、库存余额和库存流水在一个工作区内完成。
- 状态和入库模式均以中文展示，保留后端稳定状态码。
- 选择入库计划自动带出仓库、库位和经办人，减少重复录入。
- 正式入库和待检入库使用显式下拉选择，避免误把待检货物计入可用库存。
- 库存余额和流水紧贴入库明细展示，便于审批后立即核对数量一致性。
