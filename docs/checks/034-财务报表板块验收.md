## 财务报表板块能做什么

`/finance/reports` 财务统计报表页面：

- 按起始日期、结束日期、币种查询。
- 按当前筛选条件导出 CSV；六张报表均提供 `/export` 后端接口和前端导出入口。开发阶段不暴露 Excel/PDF 伪格式，避免下载内容与文件类型不一致。
- 按报表展示专项筛选：水单号、收款性质、供应商、合作伙伴、费用类型、业务员、催收状态、付款/退税状态、是否包含已登记回单。
- 页面展示统计口径、数据来源表、核心字段公式。
- 明细报表行提供来源下钻入口；能确认原始单据存在时返回对应财务页面入口。汇总统计表不展示伪下钻。
- 切换六张只读统计报表：
  - 水单使用明细：查看银行水单分摊到合同/发票的明细、分摊类型和金额。
  - 银行水单汇总：按币种汇总水单数量、总金额、已分摊、未分摊。
  - 货款支付情况：查看供应商货款付款申请、申请金额、已付、未付和状态。
  - 费用支付情况：查看合作伙伴费用付款申请、费用类型、出运单、申请金额、未付和状态。
  - 报关回单催收：查看未登记或需跟进的报关回单、提醒日期、催收状态和可退税额。
  - 申报退税统计：按币种统计核销单数、可退税额、已退税额和待退税额。
- 财务权限隔离：未登录返回 `401`，无 `finance:view` 权限返回 `403`，财务角色可访问。
- 前端有稳定路由 `/finance/reports`，并通过 E2E 验证六个 Tab、查询按钮、接口响应结构和截图。

财务模块内其它报表/查询入口：

- `/finance/receipts`：银行水单列表、收款认领/分摊、应收账款查询。
- `/finance/payments`：供应商发票、付款申请、应付账款查询。
- `/finance/fees`：合作伙伴费用发票、付费申请、应付费用查询。
- `/finance/tax`：核销单使用情况查询、报关回单登记、核销登记、退税登记。
- `/finance/port-data`：口岸数据导入、进出口报关数据查询、报关回单自动匹配核销单并回填。

## 和需求 PDF 还差哪些内容

首期功能已覆盖 PDF p54 明确列出的财务相关报表名称，但仍有几个增强项未达到传统报表系统的完整形态：

- 导出模板：当前只支持 CSV；Excel/PDF 企业套打版式、多 Sheet 工作簿、页眉页脚、水印等模板级效果尚未实现。
- 下钻体验：当前接口会校验来源单据存在并返回对应模块入口；尚未做到点击后自动定位并高亮原始行。
- 权限精细度：当前已细分 `finance:view` 和 `finance:report:export`；尚未细分到“单张报表级授权”和“敏感金额字段隐藏”。
- 报表中心整合：PDF p9/p47 还出现综合统计、出口合同/采购合同/报关出运等经理报表；这些属于 `/reporting` 经理统计分析，不属于本财务报表板块，但后续可以在报表中心统一入口整合导航。
