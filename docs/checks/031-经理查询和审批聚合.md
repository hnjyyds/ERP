# 031 经理查询和审批聚合检查记录

## PDF 对照

- `docs/00-需求范围.md 经理查询和统计分析`：要求提供待审批业务单据查询和统计分析入口。
- `docs/05-实现步骤.md 阶段 9：经理查询和统计分析`：9.1 审批查询，覆盖待审批、已审批、按单据类型、申请人和时间筛选。

## 本次实现范围

后端：

- `GET /api/v1/reporting/approvals`：聚合查询审批单据，当前覆盖出口合同和采购合同。
- 支持 `document_type`、`status`、`applicant_user_id`、`date_from`、`date_to` 查询参数。
- 返回审批单据列表、总数、待审批数、已审批数和按单据类型汇总。
- 新增 `modules/reporting/approvals` 分层模块：schema、repository、service、provider。
- 新增 `reporting:view` 权限和 `/reporting` 菜单，默认授予业务主管，不授予财务角色。

前端：

- 新增 `/reporting` React 页面。
- 页面支持按单据类型、审批状态、申请人标识、开始日期、结束日期查询。
- 首屏展示审批总数、待审批数、已审批数和单据类型数。
- 审批单据列表展示单据编号、类型、状态、客户/供应商、申请人、业务日期、提交日期、审批日期和金额。
- 类型汇总表展示各单据类型的待审批、已审批和合计数量。
