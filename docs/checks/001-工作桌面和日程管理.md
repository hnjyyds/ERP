# 001 工作桌面和日程管理检查记录

## PDF 对照

- `1.5.1 工作桌面`：公司公告、待办任务、消息管理、工作提醒、工作计划；系统定时自动刷新；提醒当前待处理事务。
- `第二部分 软件功能定制开发说明 第 7 点`：增加日程管理，每个用户登录软件后可查看自己的日程计划。

## 本次实现范围

后端：

- `GET /api/v1/health`：健康检查。
- `GET /api/v1/dashboard`：聚合公司公告、我的待办、消息提醒、我的日程、快捷入口和摘要数字。
- `POST /api/v1/schedules`：新增当前用户日程。
- SQLAlchemy async model：公告、待办、提醒、日程、快捷入口。
- Repository 返回 typed row，不向 Service 暴露无约束 dict。
- Service 负责聚合工作桌面和日程新增事务。
- FastAPI 路由显式声明 `response_model=ApiResponse[...]`。

前端：

- Vue 工作桌面页面。
- 展示公告、待办、提醒、日程、快捷入口、摘要数字。
- 支持新增日程并刷新回显。
- 支持手动刷新和 60 秒自动刷新。
- 响应式布局已通过截图检查，日程列表和表单无重叠。

## 可用性检查

- 页面采用 ERP 工作台布局，不是营销页。
- 摘要数字、待办、提醒、日程和快捷入口首屏可见。
- 日程列表与新增表单无重叠。
- 长标题可以换行，不影响表单可用性。
- 左侧导航和刷新按钮可点击。
