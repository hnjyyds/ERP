# 032 经理统计分析报表检查记录

## PDF 对照

- `docs/00-需求范围.md 经理查询和统计分析`：要求出口合同统计、采购合同统计、报关出运统计、业务员月度出货和客户出货同期对比。
- `docs/05-实现步骤.md 阶段 9：经理查询和统计分析`：9.2 统计分析报表，要求按客户、供应商、业务员、时间和状态过滤，并能下钻到原始单据。

## 本次实现范围

后端：

- `GET /api/v1/reporting/statistics`：经理统计分析聚合接口。
- 支持 `date_from`、`date_to`、`customer_id`、`supplier_id`、`sales_user_id`、`approval_status` 查询参数。
- 输出出口合同统计、采购合同统计、出货应收/利润统计、财务结算毛利汇总。
- 输出客户出货汇总、业务员月度出货、合同状态金额和原始单据下钻路径。
- 新增 `modules/reporting/statistics` 分层模块：schema、repository、service、provider。

前端：

- `/reporting` 新增“统计分析报表”工作区。
- 支持按日期、状态、客户、供应商、业务员筛选统计数据。
- 展示出口合同金额、采购合同金额、出货应收、出货利润。
- 展示客户出货表、业务员月度出货表、合同状态金额表和原始单据追溯表。
