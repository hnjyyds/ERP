# 029 杂费管理和分摊查询检查记录

## PDF 对照

- `docs/00-需求范围.md 财务管理`：杂费管理覆盖办公费用、资金占用利息、退税利息等分摊。
- `docs/05-实现步骤.md 阶段 8：财务和利润核算`：8.5 杂费管理，覆盖杂费项目配置、单票杂费分摊、分摊比例配置和杂费分摊查询。
- 本切片落地 8.5 核心闭环：财务配置杂费项目，把杂费分摊到已审批出运单，并把分摊金额作为成本进入单票利润。

## 本次实现范围

后端：

- `POST /api/v1/finance/misc-fee-items`：新增杂费项目，支持办公费用、资金占用利息、退税利息和其他杂费。
- `GET /api/v1/finance/misc-fee-items`：按关键字、分类和启停状态查询杂费项目。
- `POST /api/v1/finance/misc-fee-allocations`：把杂费项目分摊到已审批出运单，校验项目启用、出运状态和币种一致。
- `GET /api/v1/finance/misc-fee-allocations`：按关键字、分类、出运单和业务员查询分摊记录。
- `GET /api/v1/finance/misc-fee-allocations/summary`：查询分摊汇总和合计分摊金额。
- 新增 `modules/finance/misc_fees` 分层模块：model、schema、repository、service、provider。

前端：

- `/finance` 在核销退税后新增杂费管理工作区。
- 杂费项目配置支持项目编码、名称、分类、默认分摊方式、启停状态和备注。
- 单票杂费分摊支持出运单、业务员、分摊日期、金额、币种、分摊方式和分摊依据。
- 杂费分摊查询支持按关键字、分类、出运单和业务员筛选，并展示合计分摊金额。
- 移动端杂费管理区跟随财务页单列布局，避免页面级横向溢出。
